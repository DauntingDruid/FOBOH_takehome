# FOBOH Pricing App

## Project Overview

This Project allows users to:
- Filter, search, and select products
- Create/adjust pricing profiles (per product, per customer, wholesale/global, etc.)
- Apply price adjustments (fixed amount or percentage, increase or decrease)
- Expose a documented backend API via Swagger/OpenAPI
